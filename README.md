# HTML Motors üöó

**HTML Motors** is a modern front-end project built with **HTML, CSS, and JavaScript**, showcasing luxury and performance cars through a smooth animated carousel and brand-based visual themes.

The project focuses on **clean UI**, **premium animations**, and **brand identity**, using Ferrari, Porsche, and Lamborghini as visual references.

---

## ‚ú® Features

* **Animated Carousel**
  Smooth transitions between car slides with navigation arrows and indicators.

* **Brand-Based Themes**
  Each car brand has its own:
  - Color palette
  - Background gradient
  - Typography
  - Shadow styling

* **Premium UI Effects**
  - Smooth transitions and easing
  - Dynamic indicators
  - Image shadows with depth
  - Modern layout composition

* **Responsive Layout**
  Optimized for different screen sizes.

* **Pure Front-End**
  Built using only **HTML**, **CSS**, and **Vanilla JavaScript** (no frameworks).
